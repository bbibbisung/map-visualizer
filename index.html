Level Map Visualizer

Paste a full Level Blueprint (with level-json) and generate a 2D map.

1) Paste Level Blueprint Text

Paste the entire output from the Level Blueprint Generator here. The tool will automatically find the ```level-json ... ``` block.

2) Generated Map Preview

The map is drawn from the rooms and connections inside the level-json block. Room size and position are based on grid coordinates.
