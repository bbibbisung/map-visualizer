Level Map Visualizer

Paste a full Level Blueprint (with level-json) and generate a 2D map including doors, windows, props, and enemy spawns.

1) Paste Level Blueprint Text

Paste the entire output from the Level Blueprint Generator here. The tool will automatically find the ```level-json ... ``` block.

2) Generated Map Preview

The map is drawn from the rooms and detail layers (doors, windows, props, enemySpawns) inside the level-json block. Connections are still part of the JSON for gameplay logic, but are not rendered as lines in this viewer.

Layers:

Legend

  • Spawn / Start
  • Combat
  • Puzzle
  • Boss / Climax
  • Hub / Safe
  • Other types
  • Door — thick yellow tick on the room edge
  • Window — cyan dashed mark on the room edge
  • Prop / Object — sized box inside the room
  • Enemy Spawn — red circle with a crosshair
